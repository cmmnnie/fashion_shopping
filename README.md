# 🛒 Fashion Shopping - 장바구니 및 결제 모듈

## 📌 프로젝트 소개

본 프로젝트는 쇼핑몰의 핵심 기능인 **장바구니 및 결제 흐름**을 구현한 서비스입니다.

사용자는 상품을 장바구니에 담고, 수량을 조정하며,  
최종 결제 금액을 확인한 뒤 결제를 진행할 수 있습니다.

단순 기능 구현을 넘어, **컴포넌트 기반 설계와 사용자 경험(UX)**을 고려하여 개발하였습니다.

---

## 🎯 주요 기능

### 🛍 장바구니 기능
- 상품 목록 표시
- 상품 수량 증가/감소
- 상품 삭제 기능

### 💰 금액 계산 기능
- 상품 총액 자동 계산
- 배송비 자동 적용
  - 100,000원 이상 무료배송
  - 미만 시 3,000원 부과
- 최종 결제 금액 표시

### 💳 결제 기능
- 결제하기 버튼을 통한 결제 흐름 연결
- 결제 모달을 통한 결제 진행

### 🚫 예외 처리
- 장바구니 비어있을 경우 안내 문구 표시
- 결제 버튼 비활성화
- 수량 1 이하 제한
