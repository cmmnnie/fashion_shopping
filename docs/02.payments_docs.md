# 💳 결제 모듈 프로젝트

# 1. 프로젝트 개요

## 프로젝트 목적

본 프로젝트는 사용자가 상품을 장바구니에 담은 후 안전하고 편리하게 결제를 진행할 수 있도록 하는 결제 모듈을 구현하는 것을 목표로 합니다.

상품 목록 페이지에서 장바구니에 담긴 상품을 기반으로 결제 페이지로 이동하며, 사용자는 결제 수단을 선택하고 카드 정보를 입력하여 결제를 진행할 수 있습니다.

또한 사용자가 결제 과정에서 실수하지 않도록 입력값 검증과 예외 처리를 적용하여 실제 서비스 수준의 사용자 경험(UX)을 고려하여 구현하였습니다.

## 주요 기능

- 주문 상품 목록 표시
- 총 결제 금액 계산
- 결제 수단 선택
- 카드 결제 입력
- 입력값 검증
- 결제 처리
- 결제 성공 / 실패 화면 제공
