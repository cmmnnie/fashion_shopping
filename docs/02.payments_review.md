# 💳 결제 모듈 기능 리뷰

본 문서는 결제 기능이 어떻게 동작하는지 고객이 이해하기 쉽게 설명하기 위한 문서입니다.  
기능 설명은 실제 사용자 화면 기준으로 작성되었습니다.

---

# 1. 주문 상품 목록 표시

사용자가 결제 페이지에 들어오면 장바구니에 담아둔 상품이 화면에 표시됩니다.

화면에서는 다음 정보를 확인할 수 있습니다.

- 상품 이미지
- 상품 이름
- 상품 가격
- 상품 수량

사용자는 결제 전에 어떤 상품을 주문하는지 다시 확인할 수 있습니다.

동작 상태: 정상 동작 확인 완료

---

# 2. 결제 금액 표시

결제 페이지에서는 주문한 상품의 총 금액이 자동으로 계산되어 표시됩니다.

상품의 가격과 수량을 기준으로 금액이 계산되며 사용자는 최종 결제 금액을 확인할 수 있습니다.

동작 상태: 정상 동작 확인 완료

---

# 3. 결제 수단 선택

사용자는 결제 페이지에서 결제 방법을 선택할 수 있습니다.

예를 들어 다음과 같은 방식으로 결제가 가능합니다.

- 카드 결제

결제 수단을 선택하지 않으면 결제를 진행할 수 없도록 설정되어 있습니다.

동작 상태: 정상 동작 확인 완료

---

# 4. 카드 정보 입력

카드 결제를 선택하면 카드 정보를 입력할 수 있는 입력 화면이 표시됩니다.

사용자가 입력해야 하는 정보는 다음과 같습니다.

- 카드 번호
- 카드 유효기간
- CVC 번호

또한 사용자가 입력 과정에서 실수하지 않도록 다음과 같은 기능이 적용되어 있습니다.

- 카드 번호에는 숫자만 입력할 수 있습니다.
- 카드 유효기간은 올바른 범위만 입력할 수 있습니다.
- CVC 번호에는 숫자만 입력할 수 있습니다.

잘못된 값이 입력되면 결제를 진행할 수 없습니다.

동작 상태: 정상 동작 확인 완료

---

# 5. 결제 진행

사용자가 카드 정보를 입력한 후 결제하기 버튼을 누르면 결제가 진행됩니다.

결제가 진행되는 동안에는 버튼이 잠시 비활성화되어 같은 결제가 여러 번 실행되지 않도록 합니다.

동작 상태: 정상 동작 확인 완료

---

# 6. 결제 완료 화면

결제가 성공하면 결제 완료 화면이 표시됩니다.

사용자는 다음 정보를 확인할 수 있습니다.

- 결제 완료 안내 메시지
- 주문 완료 안내

동작 상태: 정상 동작 확인 완료

---

# 7. 결제 실패 안내

결제 중 문제가 발생하면 결제 실패 안내 메시지가 표시됩니다.

사용자는 다시 결제를 시도할 수 있습니다.

동작 상태: 정상 동작 확인 완료

---

# 8. 사용자 입력 오류 방지 기능

사용자가 결제 정보를 입력할 때 실수하지 않도록 다음과 같은 제한을 적용했습니다.

- 카드 번호에는 숫자만 입력 가능
- 카드 유효기간은 올바른 범위만 입력 가능
- CVC 번호는 숫자만 입력 가능

이 기능을 통해 사용자의 입력 오류를 줄이고 안정적인 결제 진행을 지원합니다.

동작 상태: 정상 동작 확인 완료
