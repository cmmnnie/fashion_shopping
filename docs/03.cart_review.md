# 장바구니 페이지 기능 안내 및 동작 확인 문서

## 1. 개요

본 문서는 장바구니 페이지에서 제공되는 기능과  
각 기능의 동작 상태를 고객사에 공유하기 위해 작성되었습니다.

사용자가 상품을 선택하고 결제까지 진행하는 과정에서  
각 기능이 정상적으로 동작하는지 확인하였습니다.

---

## 2. 기능별 설명 및 동작 상태

### 1) 상품 목록 표시

장바구니에 담긴 상품들이 목록 형태로 화면에 표시됩니다.

👉 동작 상태: 정상 작동 확인 완료

---

### 2) 수량 증가 기능

'+' 버튼을 누르면 상품 수량이 1 증가합니다.

👉 동작 상태: 정상 작동 확인 완료

---

### 3) 수량 감소 기능

'-' 버튼을 누르면 상품 수량이 1 감소하며, 1 이하로는 내려가지 않습니다.

👉 동작 상태: 정상 작동 확인 완료

---

### 4) 상품 삭제 기능

'삭제' 버튼을 누르면 해당 상품이 장바구니에서 제거됩니다.

👉 동작 상태: 정상 작동 확인 완료

---

### 5) 금액 계산 기능

상품 가격과 수량을 기반으로 금액이 자동 계산됩니다.

👉 동작 상태: 정상 작동 확인 완료

---

### 6) 배송비 계산 기능

100,000원 이상 구매 시 무료배송이 적용됩니다.

👉 동작 상태: 정상 작동 확인 완료

---

### 7) 결제하기 버튼

결제하기 버튼을 누르면 결제 화면이 나타납니다.

👉 동작 상태: 정상 작동 확인 완료

---

### 8) 결제 모달

결제 금액을 확인할 수 있는 화면이 표시됩니다.  
실제 카드 결제 기능은 아직 구현되지 않았습니다.

👉 동작 상태: UI 확인 완료 / 실제 결제 미구현

---

## 3. 종합 결과

장바구니의 주요 기능은 정상적으로 동작하는 것을 확인하였습니다.  
결제 흐름까지 자연스럽게 이어지도록 구성되어 있습니다.

다만 실제 결제 기능은 향후 추가될 예정입니다.
