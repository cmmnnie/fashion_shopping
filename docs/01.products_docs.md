# 📋 기능 목록 상세 설계 (Figma 기반 반영)

---

## 1. 상품 목록 조회 및 레이아웃 구성

### 기능 설명

- 상품 목록 페이지에서 상품 리스트를 카드 형태로 조회 및 노출
- Figma 시안 기준 Grid 레이아웃 적용
- 모바일 퍼스트 설계 후 반응형 레이아웃 확장

### 세부 작업

- 상품 데이터 준비 (Mock 데이터 또는 API 연동)
- 상품 리스트 렌더링 처리
- Grid 기반 레이아웃 구성
- 반응형 컬럼 수 제어  
  - 모바일: 2열  
  - 태블릿: 3열  
  - 데스크탑: 4열
- 브레이크포인트 정의  
  - ≤ 768px  
  - 768px ~ 1024px  
  - ≥ 1024px

### 사용 컴포넌트

- `<ProductList />`

---

## 2. 상품 카드 UI 구성

### 기능 설명

- Figma 시안과 동일한 카드 UI 구조 적용
- 상품 이미지, 상품명, 가격 정보 노출
- 카드 클릭 시 상세 페이지 이동

### 세부 작업

- 상품 이미지 렌더링 (비율 유지)
- 상품명 텍스트 스타일 적용
- 가격 포맷 적용 (₩ 표기 및 천단위 콤마)
- 카드 클릭 이벤트 처리
- Hover / Active 상태 UI 반영

### 사용 컴포넌트

- `<ProductCard />`

## 3. 상품 담기 버튼 및 상태 UI

### 기능 설명

- 상품 카드 내 장바구니 담기 버튼 제공
- 상품 담김 여부에 따른 버튼 상태 변경
- 담김 시 시각적 피드백 제공

### 세부 작업

- 담기 버튼 클릭 이벤트 처리
- 상품 담김 상태 관리
- 담김 / 미담김 상태에 따른 버튼 UI 변경
- 버튼 비활성화 상태 처리

### 사용 컴포넌트

- `<ProductCard />`
- `<AddToCartButton />`
